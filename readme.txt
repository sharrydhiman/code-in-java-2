# ArrayToArrayList Converter

This Java program allows you to convert an 
array into an ArrayList and vice versa. 
It takes user input for creating an array 
and an ArrayList, and then performs the conversions.

## Usage



1. **Compile and Run:**
    ```bash
    javac ArrayToArrayList.java
    java ArrayToArrayList
    ```

2. **Follow the Prompts:**
    - Enter the size of the array.
    - Enter elements for the array.
    - Enter the size of the ArrayList.
    - Enter elements for the ArrayList.

3. **Output:**
    - The program will display the array converted to an ArrayList.
    - The program will display the ArrayList converted to an array.

## Example

```java
// Example array conversion
Enter the size of the array:
5
Enter elements for the array:
Element 1: 10
Element 2: 20
Element 3: 30
Element 4: 40
Element 5: 50
Array converted to ArrayList: [10, 20, 30, 40, 50]

// Example ArrayList conversion
Enter the size of the ArrayList:
3
Enter elements for the ArrayList:
Element 1: apple
Element 2: banana
Element 3: orange
ArrayList converted to Array: [apple, banana, orange]
